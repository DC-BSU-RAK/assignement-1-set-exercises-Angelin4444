Code Navigator: HTML & CSS Guide

This is a guide that covers the mostly used HTML tags, attributes, and CSS properties, grouped together into easy-to easy-to-read sections with practical examples for better understanding.

What is HTML ?

HTML stands for Hyper Text Markup Language. HTML is the code that is used to structure a web page and its content.

Lets dive into some of the 'HTML tags'..

This is the basic structure of a HTML document. A HTML consist of two parts. "Head" and a "Body". First it starts with Html tag opened. Then Head is opened where all the meta data(All the details in head will not be shown in the website)is mentioned. After closing head, Body is opened and all the details in the body part is shown in the website. Then the body tag is closed along with the html tag.

Basic HTML Structure

<html>
      <head>
        <title>Page Title</title>
      </head>
      <body>
        Page content...
      </body>
</html>

These are some of the commonly used 'Tags' in a HTML document.

Common Tags

  • <h1> to <h6> – Used for 'Headings' (h1 for largest to h6 for smallest)
  • <p> – 'Paragraph tag' used for writing sentences and paragrahs
  • <b>, <strong> – It is used to get 'Bold text'
  • <i>, <em> – This one is for 'Italic text'
  • <a href="url"> – Create a 'link' to another page or website(basically used for getting clickable links)
  • <img src="image.jpg" alt="description"> – This helps in showing 'Images' in the website
  • <br> – It is used for break 'Lines' in between
  • <ul><li></li></ul> – Helps in creating Unordered List (Basically Bullet-points list).

These are some of the 'Tags' used for text formatting.

Text Formatting

  • <u> – 'u' is a tag used for underlining words or text
  • <strike> – This tag 'Strikethrough' words or text
  • <sup> – 'Superscript' helps to get smaller text placed 'above' normal text
  • <sub> – 'Subscript' helps to get smaller text placed 'below' normal text
  • <small/big> – Used to get small/big text
  • <tt> – To get 'Typewriter' effect for the Text

These are for 'Section Division'.

Section Divisions

  • <div/section> – 'Division' or 'Section' of the Page Content
  • <hr> – This is used for getting basic 'horizontal line'/li>
  • <nobr> – Used for 'line breaking'

Next section talks about 'Media'.

Media

  • <audio controls src=""audio> – This helps in showing 'Audios' in the website
  • <video src=""video> – This helps in showing 'Videos' in the website /li>

What is CSS ?

CSS stands for Cascading Style Sheet. It is a language that is used to describe the style of a document.

Lets dive into some of the 'CSS tags'..

There are three ways to include CSS into HTML document

Inline CSS

  • <h1 style=""></h1> – Here the styling is done within the lines of html tags

Internal CSS

  • <style> h1 { color: blue; } </style> – Here the styling is written inside a style tag and it is placed within the head of the HTML document.

External CSS

  • <link rel="stylesheet" href="styles.css"> – This opens up a external CSS file to the HTML document. The CSS rules are written in a separate file (like styles.css).

CSS Properties

  • color – Used it for 'color'
  • background-color – Used for 'Background color'
  • font-size – 'Size' of text is selected
  • margin – Make sure of Space outside element(no text go outside the page )
  • padding – Makes sure of Space inside element(no text overlap each other or touches the margin)
  • border – Helps in creating 'border'
  • text-align – Makes sure that the 'alignment' of text is proper
  • display – Ensures the 'Layout' of the page
